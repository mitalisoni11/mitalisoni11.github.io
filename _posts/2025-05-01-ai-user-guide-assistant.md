## 🚀 Project Overview

Many B2B SaaS users struggle to find the right answers in lengthy product documentation. This AI-powered assistant helps users find instant, context-aware answers from user guides using **Retrieval-Augmented Generation (RAG)** — dramatically improving support efficiency and user satisfaction.

By integrating OpenAI and Pinecone, this project offers a responsive and intelligent support interface that reduces ticket volume and boosts self-service resolution.

## ✨ Key Features

- 💬 **Interactive Chat UI**: Users can ask questions naturally and receive helpful, AI-generated answers.
- 🔍 **RAG-Powered Smart Search**: Semantic search retrieves the most relevant sections from documentation before generating answers.
- 🖼️ **Image & Flowchart Integration**: Includes relevant screenshots and visuals directly in the response.
- 🔄 **Auto-Context Switching**: Keeps track of the user’s question flow for contextual responses.

## 🧠 Tech Stack

- **Backend**: Flask, OpenAI API, Pinecone
- **Frontend**: Streamlit (for interactive UI)
- **Data Processing**: pdfplumber (PDF extraction), Pillow (image handling)
- **Vector Search**: Pinecone (semantic search & retrieval)
